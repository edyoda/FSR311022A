Tags in HTML 
<title></title> => Used to show the title of web page on the tab of the browser

Heading
<h1></h1> => Used to write the heading on a page (Largest heading)
<h2></h2>
<h3></h3>
<h4></h4>
<h5></h5>
<h6></h6> => Smallest heading

<p></p> => Paragraph Tag (Used to write decriptive text/paragraph)


Attributes => These are the options which can be given to a tag to provide extra features.

<link> => Used to link CSS files
<script> => Used to link JS files

<meta> => Data about data (Any information that describes our web page) (SEO (Search Engine Optimization))
